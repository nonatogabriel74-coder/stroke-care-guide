# Comparação entre AVC Isquémico e AVC Hemorrágico

| Critério    | AVC Isquémico                      | AVC Hemorrágico                   |
| ----------- | ---------------------------------- | --------------------------------- |
| Causa       | Obstrução arterial                 | Ruptura de vaso sanguíneo         |
| Frequência  | Mais comum                         | Menos comum                       |
| Diagnóstico | TC e RM                            | TC e RM                           |
| Tratamento  | Trombolíticos e controlo clínico   | Controlo da hemorragia e cirurgia |
| Prognóstico | Melhor quando tratado precocemente | Maior risco de complicações       |

## Observação

O reconhecimento precoce dos sintomas é essencial para aumentar as probabilidades de recuperação.
