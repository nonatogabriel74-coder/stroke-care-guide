# Metodologia do Projeto

## Objetivo

Desenvolver uma base de conhecimento estruturada sobre prevenção, tratamento e reabilitação de pacientes com AVC utilizando NotebookLM.

## Etapas Realizadas

### 1. Seleção das Fontes

Foram escolhidos documentos de instituições reconhecidas e com elevado rigor científico.

### 2. Carregamento no NotebookLM

Os documentos foram importados para criar uma base de conhecimento especializada.

### 3. Engenharia de Prompts

Foram desenvolvidos prompts específicos para explorar diferentes aspetos do AVC.

### 4. Análise dos Resultados

As respostas geradas foram avaliadas quanto à precisão, relevância e alinhamento com as fontes.

### 5. Documentação

Todos os resultados, aprendizagens e desafios foram registados neste repositório.
