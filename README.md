# Comprehensive Stroke Care Guide: Prevention, Treatment, and Rehabilitation

## Sobre o Projeto

Este repositório documenta um projeto de curadoria e síntese de informações médicas utilizando o **NotebookLM**. O objetivo é transformar documentos técnicos complexos sobre AVC em um guia prático, útil tanto para estudantes de saúde quanto para cuidadores domiciliares. O projeto foca em **Tradução de Conhecimento**, garantindo que a informação científica seja acessível, segura e humanizada.

## 🎯 Objetivo do Projeto

Este projeto nasceu de uma experiência pessoal ao observar de perto o impacto do Acidente Vascular Cerebral (AVC) na vida de um familiar próximo, num contexto onde o acesso a cuidados de saúde dignos e informação médica clara ainda representa um desafio significativo para muitos cidadãos.

A partir dessa realidade, o objetivo deste trabalho é criar um sistema de conhecimento estruturado que facilite a compreensão do AVC e apoie decisões informadas em contexto clínico e domiciliário.

Através da utilização de Inteligência Artificial (NotebookLM), o projeto procura reduzir a complexidade da informação médica, transformando conteúdos técnicos em conhecimento acessível, organizado e útil para cuidadores, estudantes e profissionais de saúde.

Mais do que um exercício técnico, este projeto representa um esforço para aproximar o conhecimento médico das pessoas que dele mais necessitam.

## Estrutura do Guia Final

O conteúdo gerado organiza-se em:

1. **Módulo Técnico:** Protocolos de reabilitação neurofuncional.
2. **Manual do Cuidador:** Rotina de assistência domiciliar.
3. **Matriz de Resposta Rápida:** Sinais de alerta (Tabela).

## 👨‍⚕️ Manual do Cuidador

- Rotina de assistência domiciliar
- Apoio à alimentação e mobilidade
- Comunicação com o paciente
- Suporte emocional e psicológico

## ⚠️ Matriz de Resposta Rápida

Tabela de sinais de alerta para identificação precoce de AVC:

| Sinal | Descrição |
|------|----------|
| Fraqueza súbita | Perda de força em um lado do corpo |
| Dificuldade na fala | Fala arrastada ou incoerente |
| Assimetria facial | Um lado do rosto caído |
| Perda de visão | Alteração súbita na visão |
